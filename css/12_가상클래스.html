# Pseudo class

  • 선택자 뒤에 추가해 해당 요소의 특정 상황만을 선택하는 것
  • 선택자 뒤에 :을 붙여 원하는 특정 상황만을 선택할 수 있다

# 종류

  • :first-child - 해당 요소가 첫 번째 자식인 상황에 적용할 스타일
  • :first-of-type - ...
  • :hover - 마우스가 해당 요소 위에 올려져있는 상황
  • :link - 해당 링크가 방문한적 없는 링크인 상황
  • :active - 해당 링크를 클릭
  • :visited - 해당 링크에 방문한 적이 있는 상황
