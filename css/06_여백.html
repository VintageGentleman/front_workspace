# HTML Box Model

  • HTML의 모든 요소는 박스 형태로 구성되어 있다
  • 요소를 구성하는 박스의 제일 바깥쪽 여백이 margin이고 border, padding, content 순으로 요소를 박스형태로 구성하게 된다.
  • F12 개발자 도구를 통해 그림으로 확인할 수 있다

# box-sizing

box-sizing: content-box
width/height property가 content-box의 크기를 의미하게 된다
box-sizing: border-box
width/height property가 border-box의 크기를 의미하게 된다
크기 500px로 설정한 content-box
(요소의 크기 = width/height + padding + border)
크기 500px로 설정한 border-box
(요소의 크기 = width/height)
