# input 태그를 사용해 사용자로부터 다양한 입력을 받는다

Type Example Description Usage
type="text" 일반적인 텍스트 입력란 아이디 입력, 글 제목 등
type="password" 입력하면 *로 나오는 입력란 회원 비밀번호, 글 비밀번호 등
type="number" 숫자 입력란 수량 등
type="date" 날짜 선택란 예약일자 등
type="radio"



같은 카테고리의 옵션 중 하나만 선택해야하는 버튼 성별 등
type="checkbox"


같은 카테고리 내에서 복수선택이 가능한 버튼 설문조사 등
<textarea> 긴 내용의 텍스트를 입력받는 태그 (input 아님) 글 내용 등
<select> 여러 옵션 중 하나를 선택하는 태그 국가 선택 등
