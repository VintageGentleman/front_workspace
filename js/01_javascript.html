# Javascript

  • 웹 점유율 100%의 프로그래밍 언어
  • 웹 브라우저가 해석하여 실행시킬 수 있는 프로그래밍 언어
  • 최근에는 언어 자체의 성능이 좋아져서 다른 용도로도 많이 사용된다 (구글이 만든 크롬 내부의 자바 스크립트 해석 성능(V8 엔진)이 너무 좋아서 node.js 등을 만들 수 있었다)

# Javascript가 할 수 있는 일

  • 웹 페이지 요소의 스타일을 변경한다
  • 웹 페이지 요소에 이벤트를 추가한다
  • 웹 페이지에 새로운 요소를 추가/수정/삭제한다

# 웹 페이지가 해석되는 순서

  1. 브라우저가 웹 코드를 통해 HTML 구조를 먼저 계층적으로 정리한다 그 결과로 DOM(Document Object Model)이 생성된다.
  2. 화면에 그릴 순서를 정리한다
  3. 각 요소의 CSS 스타일을 결정한다
  4. 레이아웃을 설정하고 그린다
  5. 위의 과정 도중 자바스크립트를 만나면 해당 과정을 일시 중단하고 자바스크립트를 먼저 실행한다.
